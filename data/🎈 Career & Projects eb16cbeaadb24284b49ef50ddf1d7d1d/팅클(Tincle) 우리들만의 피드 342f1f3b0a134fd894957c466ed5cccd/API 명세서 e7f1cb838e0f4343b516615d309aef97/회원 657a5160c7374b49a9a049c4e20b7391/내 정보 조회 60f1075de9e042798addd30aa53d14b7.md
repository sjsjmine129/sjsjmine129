# 내 정보 조회

URL Patterns: /accounts/me
담당자: 영두 우
Method: GET
작업 완료: Yes
1차 MVP: Yes

## GET /api/account/me

---

## **Request**

- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    

## Response

- **Success (200) : 조회 성공**
    
    ```json
    {
        "data": {
            "accountId": 51,
            "nickname": "우영두",
            "status": "SMILE"
        }
    }
    ```
    
- **Error:**
    - **404: 회원이 존재하지 않음**
    
    ```json
    {
        "statusCode": 2010,
        "data": {},
        "message": "not found account error."
    }
    ```
