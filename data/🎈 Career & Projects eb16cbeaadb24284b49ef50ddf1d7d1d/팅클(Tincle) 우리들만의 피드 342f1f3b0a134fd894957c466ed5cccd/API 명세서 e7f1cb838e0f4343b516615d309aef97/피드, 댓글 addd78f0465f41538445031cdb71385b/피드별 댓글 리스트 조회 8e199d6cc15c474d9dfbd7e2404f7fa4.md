# 피드별 댓글 리스트 조회

URL Patterns: /feeds/{feedId}/comments
담당자: 영두 우
Method: GET
작업 완료: Yes
1차 MVP: Yes

## GET /feeds/{feedId}/comments/parents

---

## **Request**

ex. `/feeds/{feedId}/comments?cursorId=0`

- **Query**
    
    
    |  | 설명 | 타입 | 예시 |
    | --- | --- | --- | --- |
    | cursorId | 커서 id
    default : 0 | Integer |  |
    | size | 개수 제한
    default : 10 | Integer |  |
    
- **Params**
    - feedId (필수)
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
