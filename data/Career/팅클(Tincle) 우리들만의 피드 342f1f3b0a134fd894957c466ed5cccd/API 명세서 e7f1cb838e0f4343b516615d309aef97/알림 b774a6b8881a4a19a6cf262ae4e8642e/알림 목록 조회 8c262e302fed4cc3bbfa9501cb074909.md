# 알림 목록 조회

URL Patterns: /notifications/accounts/me
담당자: 영두 우
Method: GET
작업 완료: No

## GET /notification/accounts/me

---

- targetEntity : 해당 알림을 클릭했을 때 넘어가야할 대상
- redirectTargetId : 해당 알림을 클릭했을 때 넘어가야할 대상 id

## **Request**

- **query**
    - cursorId - int
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
