# 피드 삭제

URL Patterns: /feeds/{feedId}
담당자: 영두 우
Method: DELETE
작업 완료: Yes
1차 MVP: Yes

## DELETE /feeds/{feedId}

---

## **Request**

- **Params**
    - feedId (필수)
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    

## Response

- **Success (200) :**
    
    ```json
    {
        "data": {
            "result": true
        }
    }
    ```
    
- **Error:**
    - **403 : 작성자가 아닌 경우**
    
    ```json
    {
        "statusCode": 1500,
        "data": {},
        "message": "not author error"
    }
    ```
    
    - **404: 피드가 존재하지 않음 (param이 잘못됨)**
    
    ```json
    {
        "statusCode": 4030,
        "data": {},
        "message": "not found feed error."
    }
    ```
