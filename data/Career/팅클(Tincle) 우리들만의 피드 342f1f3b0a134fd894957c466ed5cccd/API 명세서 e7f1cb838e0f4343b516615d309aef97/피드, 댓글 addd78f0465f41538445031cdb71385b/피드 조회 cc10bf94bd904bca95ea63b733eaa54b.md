# 피드 조회

URL Patterns: /feeds
담당자: 영두 우
Method: GET
작업 완료: Yes
1차 MVP: Yes

## GET /feeds

---

## **Request**

ex. `/feeds?cursorId=0`

- **Query**
    
    
    |  | 설명 | 타입 | 예시 |
    | --- | --- | --- | --- |
    | cursorId | 커서 id
    default : 0 | Integer |  |
    | size | 개수 제한
    default : 10 | Integer |  |
    
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
