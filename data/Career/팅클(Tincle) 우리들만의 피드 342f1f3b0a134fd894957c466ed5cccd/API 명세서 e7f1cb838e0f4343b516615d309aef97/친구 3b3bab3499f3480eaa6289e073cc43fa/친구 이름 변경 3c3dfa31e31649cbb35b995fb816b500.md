# 친구 이름 변경

URL Patterns: /friendships/nickname/change
담당자: 영두 우
Method: POST
작업 완료: Yes
1차 MVP: Yes

## POST /friends/nickname/change

## **Request**

- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    
- **Body**
    
    ```json
    {
    		"friendAccountId": 1
        "nickname": "영두"
    }
    ```
    

## Response

- **Success (200) :**
    
    ```json
    {
        "data": {
            "friendNickname" : "영두"
        }
    }
    ```
    
- **Error:**
    - 404 : accountId를 찾을 수 없을 경우
    
    ```json
    {
        "statusCode": 2010,
        "data": {},
        "message": "not found accountId error."
    }
    ```
