작업 완료: Yes

## POST /auth/signup

---

로그인 접근 시 발급 받은 SignToken 정보 필요

## **Request**

- **Body**
    
    ```json
    {
      "signToken" : String,
    	"usePolicy" : true,
    	"agePolicy": true,
    	"personalPolicy": true,
    	"marketPolicy": true,
    	"fcmToken": String,
    	"code" : String
    }
    ```
    
    - `signToken` : 이메일, 회원 정보가 들어 있는 JWT 토큰 값
    - `fcmToken`: fcm 토큰 입력
    - `usePolicy`: 이용약관 및 개인정보취급방침
    - `agePolicy`: 만 14세 이상 확인 (필수)
    - `personalPolicy`: 개인정보 수집 이용 동의 (필수)
    - `marketPolicy`: 마케팅 활용 동의 (선택)
    - `code` : 유니크한 아이디
