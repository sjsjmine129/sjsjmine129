# 프로필 업데이트(status)

URL Patterns: /accounts/me/status/{status}
담당자: 영두 우
Method: PUT
작업 완료: Yes
1차 MVP: Yes

## POST /accounts/me/profile/status

---

프로필 업데이트

status를 수정할 수 있음

## **Request**

- **Params**
    - status 필수
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    

## Response

- **Success (200) : 변경완료**
    
    ```json
    {
        "data": {
            "status": "READ"
        }
    }
    ```
    
- **Error:**
    - **404: status가 잘못됐을 때**
    
    ```json
    {
        "statusCode": 2050,
        "data": {},
        "message": "not found status error"
    }
    ```
    
    - **400: 현재 status와 바꾸려는 status가 같을 때**
    
    ```json
    {
        "statusCode": 2060,
        "data": {},
        "message": "reqeust status equal account's status"
    }
    ```
