# 익명 쪽지 쓰기

URL Patterns: /accounts/{accountId}/message
담당자: 영두 우
Method: POST
작업 완료: Yes
1차 MVP: Yes

## POST /accounts/{accountId}/message

---

## **Request**

- **Params**
    - accountId 필수
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    
- **Body**
    
    ```json
    {
    		"message" : "너 누구야"
    }
    ```
    

## Response

- **Success (200) :**
    
    ```json
    {
        "data": {
            "messageBoxId": 3,
    				"message": "너 누구야",
    				"createdAt" : "오늘"
        }
    }
    ```
