# 댓글/대댓글 수정

URL Patterns: /feeds/comments/{commentId}
담당자: 영두 우
Method: PUT
작업 완료: No
1차 MVP: No

## PUT feeds/comments/{commentId}

---

## **Request**

- **Params**
    - commentId 필수 필수
- **Header**
    
    ```json
    Authorization : Bearer JWT
    ```
    
- **Body**
    
    ```json
    {
        "content": "댓글 내용"
    }
    ```
    

## Response

- **Success (200) :**
    
    ```json
    {
        "data": {
    	    "parentId": 30,
    	    "childCount": 0,
    	    "commentId": 35,
    	    "content": "string",
    	    "createdAt": "string",
    	    "accountId": 0,
    	    "nickname": "string",
    	    "statusImageUrl": "string",
    	    "isAuthor": true
    	  },
    }
    ```
    
    - parentId: 부모댓글이면 null값
    - childCount : 대댓글 수
